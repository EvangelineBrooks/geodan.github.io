Demo of the jQuery / Bootstrap Dual List Box

In this demo a single select will be created and filled through a jQuery JSON request. After which the Dual List Box will be created and filled with the given values. One can also opt to define already selected <option>; elements and provide them to the Dual List Box as well. Downloads and more information can be found on the GitHub repository.
